# Chapter 3 드디어 LED를 깜박이다

## 1. 포인터 없이 메모리에 접근하기
메모리 접근에 포인터가 꼭 필요한 것은 아니다.

**직접 접근** : 포인터 없이 메모리에 접근하는 방법
    
    *(volatile unsigned int *)0x3000000 = 100;
    

메모리 레이아웃이 어떻게 구성되어 있는지 정확하게 파악하지 않은 상황에서 사용하면 안된다. 직접 접근 방법은 포인터를 통한 간접 접근 보다 빠르다.
